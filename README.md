# 股票即時爬蟲看板

這是一個基於 Python 和 Streamlit 的股票即時監控應用程式。它可以讓你輸入股票代號（台股或美股），並自動抓取最新的報價與漲跌幅，且支援自動定時刷新。

## 功能特色
- **支援多國股市**：可輸入台股（如 `2330.TW`）或美股（如 `AAPL`, `NVDA`）。
- **即時連動**：設定刷新頻率，自動獲取最新數據。
- **視覺化看板**：清楚顯示最新價、漲跌幅，並以顏色區分（紅漲綠跌）。
- **彈性擴充**：核心爬蟲邏輯獨立，未來可擴充更多資料來源。

## 安裝說明

### 1. 安裝 Python
如果您尚未安裝 Python，請先前往 [Python 官網](https://www.python.org/downloads/) 下載並安裝最新版本（建議 3.8 以上）。
**注意：安裝時請勾選 "Add Python to PATH" 選項。**

### 2. 安裝依賴套件
在專案目錄下開啟終端機 (Terminal) 或命令提示字元 (CMD)，執行以下指令：

```bash
pip install -r requirements.txt
```

### 3. 啟動應用程式
執行以下指令來啟動看板：

```bash
streamlit run app.py
```

啟動後，瀏覽器會自動開啟，或者您可以點擊終端機中顯示的連結（通常是 `http://localhost:8501`）。

## 使用方法
1. 在左側欄位輸入您想監控的股票代號，使用逗號分隔。
   - 台股範例：`2330.TW, 0050.TW, 2603.TW`
   - 美股範例：`AAPL, TSLA, MSFT`
2. 調整「刷新頻率」滑桿，設定多久更新一次數據。
3. 右側主畫面將會顯示即時報價卡片與詳細表格。

## 檔案結構
- `app.py`: 主程式，包含 Streamlit 介面邏輯。
- `crawler.py`: 爬蟲邏輯，負責抓取股票數據 (目前使用 yfinance)。
- `requirements.txt`: 專案所需的 Python 套件列表。
